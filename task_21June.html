FLEX

This is the shorthand for flex-grow, flex-shrink and flex-basis combined. The second and third parameters are optional. Default is 0 1 auto.

FLEX GROW

This defines the ability for a flex item to grow if necessary. It dictates what amount of available space inside the flex container the item should take up.

FLEX SHRINK

This defines the ability for a flex item to shrink if necessary. Negative numbers are invalid.
