What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and design the structure of web pages. It consists of elements represented by tags that define the content and layout of the web page.

What are HTML tags? Give examples.

HTML tags are code snippets enclosed within angle brackets (< >) that define elements on a webpage. Tags typically come in pairs: opening tag and closing tag .

What is the purpose of the tag in an HTML document?

The tag is used to contain meta-information about the HTML document, such as its title, character set, and links to stylesheets or scripts. This information is not displayed directly on the web page but is essential for the browser and search engines.

What is the purpose of the tag in an HTML document?

The tag is used to contain the main content of the HTML document, including text, images, links, and other media. This content is displayed in the browser window and is what users interact with on the web page.
